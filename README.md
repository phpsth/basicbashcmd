# This package is show you how to use bash command

PyPi: https://pypi.org/project/basicbashcmd

## Installation

Launch CMD / Terminal

```python
pip install basicbashcmd
```

## Usage

- Launch IDLE and type...

```python
from basicbash import Bash
b = Bash()
b.list()
b.desc('cd')
b.cmd('cp')
```
